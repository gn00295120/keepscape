# Judging evidence map

This file keeps implementation priorities tied to the four equally weighted Build Week criteria.

| Criterion | Claim | Evidence judges can inspect |
| --- | --- | --- |
| Technological implementation | GPT-5.6 emits a strict source-linked blueprint containing no geometry; the host compiles a safe diorama and converts the reviewed structure to opaque tokens; Codex returns an enum-only mechanic/spatial plan with no prose; the host rebinds exact sets, preserves copy and sequence order, applies canonical slots, and validates references and schema. | `/api/blueprint`, `/api/build`, `createOpaqueCodexProjection`, dynamic token-enum schema, exact-set/reorder tests, `createPhotoDiorama`, post-build schema test, actual trace and receipt. |
| Design | Keepscape is a complete archive-to-walkable-space journey, not a prompt box or isolated demo. | Source desk, uncertainty review, build transition, multi-photo diorama, Evidence Lens, second mechanic, responsive and keyboard paths. |
| Potential impact | Families can preserve the context behind ordinary photos without surrendering truth to generative fill-in. | Original-voice timecodes, cited photo views, manually calibrated fixture regions, text-only notes that never masquerade as audio, typed human decisions, generated-language review, no-cloning policy. |
| Quality of idea | Generated space and provenance coexist inside the emotional experience. | Evidence Lens preserves the selected cited source view and, when reviewed metadata exists, outlines its region; Lantern and Repair share one contract but produce different play. |

## Demo proof order

1. Turn three source photographs directly into a full-screen walkable memory corridor.
2. Turn on Evidence Lens so the Truth Thread connects a detail to its source crop, audio timecode, and human
   decision in one frame.
3. Preserve one unsupported claim as uncertain instead of letting the model promote it to fact.
4. Show the GPT-5.6 / person / Codex roles, then the real public replay and pinned live SDK receipt.
5. Complete Lantern Lane and hold on **3/3 · archive awakened**.
6. Complete the evidence-shaped Repair Bench sequence and hold on **4/4 · archive awakened**.
7. Show the public engineering proof card, then close on the product promise: AI can build the place; it cannot
   rewrite the memory.

The public judge path is a verified deterministic replay with its fallback trace shown in-product. It proves the
runtime and validation journey without implying that GPT-5.6 or Codex ran live during that browser session.
